# 项目架构
后端：

+ SpringBoot
+ JWT
+ Sa-token
+ MySQL
+ 图片存储：Mongodb（暂定）

前端：

+ 客户端：
    - Flutter
    - Dio
    - GetX
+ Admain端：
    - Vue3（之前学的Vue2，不是很熟悉Vue3的特性，尤其是ts）
    - Vite（打包工具）
    - ElementPlus
